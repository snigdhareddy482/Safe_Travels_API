# Testing Module

This directory contains all test suites for the SafeTravels application.

## Purpose
- To ensure code quality and reliability through automated testing.

## Structure
- `unit/`: Unit tests for individual components.
- `integration/`: Integration tests for API endpoints and database interactions.
- `e2e/`: End-to-end tests for full system flows.
